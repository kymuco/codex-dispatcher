# Security Policy

## Supported Versions

Security updates are currently provided for:

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Use one of these paths:

1. Open a private GitHub Security Advisory report for this repository.
2. If private advisory reporting is unavailable, open an issue with minimal detail and request a private contact channel.

## Response Targets

- Initial acknowledgement: within 72 hours.
- Triage and severity assessment: within 7 days.
- Fix timeline depends on impact and complexity.

## Disclosure

After a fix is available, we will publish a changelog note and release details.
